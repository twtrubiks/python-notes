## 其他

如果你不是用 docker 建立 redis, 是使用安裝在本機上的,

設定開機自動啟動 redis, 可以透過 [systemctl](https://github.com/twtrubiks/linux-note/tree/master/systemctl-tutorial),

```cmd
sudo systemctl enable redis-server
```
